# Esports Match

School project to create a website listing upcoming, ongoing, and completed esports matches for various competitive games.

## Project Status

Project under development.

## Objectives

- Display a list of esports matches (completed, ongoing, scheduled)
- Filter by game, date, or team
- View match details
- Simple and responsive interface

## Technologies Used

### Front-end
- HTML
- CSS
- TypeScript

### Back-end
- C#

### Data
- MongoDB

## Author

Project completed as part of a school assignment.
